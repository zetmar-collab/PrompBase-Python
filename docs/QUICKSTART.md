# PrompBase — szybki start (2 minuty)

## 1. Pobierz i uruchom

1. Pobierz **PrompBase.exe** z [GitHub Releases](https://github.com/zetmar-collab/PrompBase-Python/releases).
2. Kliknij dwukrotnie — bez instalacji Pythona.
3. Przejdź **przewodnik startowy** (pierwsze uruchomienie).

## 2. Pierwsza akcja (30 sekund)

1. Wybierz przykładowy prompt z listy (np. *Post LinkedIn — marketing*).
2. Kliknij **Kopiuj i użyj w AI** (lub `Ctrl+Shift+A`).
3. Wklej treść w otwartym ChatGPT / Claude (`Ctrl+V`).

## 3. Własny prompt

- **+ Nowy Prompt** — nazwa, model, treść.
- Zmienne w treści: `[TEMAT]`, `[PRODUKT]` — aplikacja zapyta przy kopiowaniu.

## 4. Gdzie są dane

Windows: `%APPDATA%\PrompBase\promptbase.json`  
Kopia zapasowa: `promptbase.json.bak`

## Skróty

| Skrót | Akcja |
|-------|--------|
| `Ctrl+Shift+A` | Kopiuj i użyj w AI |
| `N` / `Ctrl+N` | Nowy prompt |
| `/` lub `Ctrl+F` | Szukaj |
| `?` | Pomoc |
