FAQ

Czy potrzebuję Pythona?

Nie. Pobierz PrompBase.exe z GitHub Releases — działa od razu na Windows 10/11.

Gdzie są moje dane?

Windows: %APPDATA%\PrompBase\promptbase.json. Kopia zapasowa: promptbase.json.bak.

Czy działa na Mac?

Tak — wersja ze źródeł (Python + Tkinter). EXE jest tylko dla Windows.

Mam starą wersję w przeglądarce (PWA)?

W aplikacji: Plik → Import z PWA. Szczegóły w dokumentacji na GitHub.
