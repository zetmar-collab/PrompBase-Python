## PrompBase Python 2.5

### Sprzedaż i UX (CRO)

- Przewodnik startowy przy pierwszym uruchomieniu
- Checklista „Pierwsze kroki” w panelu bocznym
- Główny przycisk **Kopiuj i użyj w AI** (Ctrl+Shift+A)
- Przykłady: marketing, kod Python, n8n
- Empty state: „Załaduj przykłady” / „+ Nowy Prompt”
- Motyw **grafit** + skróty w menu Pomoc (strona produktu, pobieranie)
- Landing: `landing/index.html`

### Dokumentacja

- README i OPIS-NAFFY pod korzyści i jedno CTA
- PRICING.md (Free + plan Pro w przygotowaniu)

### Paczka dystrybucyjna

`scripts/build_windows.ps1` tworzy:

- `dist/PrompBase.exe`
- `dist/PrompBase-2.5-Windows.zip` (EXE + landing + docs + START.txt)
