## PrompBase Python 2.4

Lokalna biblioteka promptów AI dla Windows i macOS.

### Pobierz

- **PrompBase.exe** (Windows, ~12 MB) — uruchom bez instalacji Pythona
- Lub klonuj repo i użyj `Start-PrompBase-Windows.bat` / `python promptbase.py`

### Nowości

- Lista modeli AI 2026 (GPT-5.5, Claude Opus 4.7, Gemini 3.5…)
- Przyciski: ChatGPT, Claude, Gemini, Perplexity, Copilot
- Import/eksport PWA (`promptLibrary` z localStorage)
- Eksport do folderu Google Drive / OneDrive
- Historia wersji promptu (5 zapisów)
- Tagi, zmienne `[TEMAT]`, podświetlanie wyszukiwania
- Integracja n8n (test, single, sync)
- Kopia zapasowa JSON + automatyczny `.bak`

### Dane użytkownika

Windows: `%APPDATA%\PrompBase\promptbase.json`
