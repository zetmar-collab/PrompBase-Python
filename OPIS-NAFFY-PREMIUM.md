## Długi opis (wklej na naffy)

Masz ChatGPT, Claude albo Gemini — ale **ciągle piszesz te same polecenia od zera** albo szukasz ich w notatkach z sprzed miesiąca?

**PrompBase Premium** to paczka **250 sprawdzonych promptów po polsku**, przygotowanych pod realną pracę: kampanie, maile, landingi, kod, debug, shorty na Reels i TikTok. Nie dostajesz luźnej listy zdań — każdy prompt ma strukturę, którą lubią modele AI: rola, kontekst, zadanie, format odpowiedzi i zasady.

### Co zyskujesz

- **Oszczędzasz godziny** — zamiast wymyślać prompt, wybierasz gotowy, uzupełniasz 2–3 pola i klikasz „Kopiuj”.
- **Lepsze odpowiedzi AI** — prompty są ułożone jak u specjalisty od prompt engineeringu (nagłówki, placeholdery, oczekiwany format wyniku).
- **Wszystko po polsku** — copy, maile, strategie, scenariusze video, code review, API, DevOps.
- **Import w minutę** — jeden plik JSON do darmowej aplikacji PrompBase (Windows). Bez konta, bez chmury.

### Dla kogo

- **Marketerzy i właściciele małych firm** — persony, lejki, maile, landingi, reklamy, SEO.
- **Programiści i freelancerzy IT** — review kodu, API, testy, DevOps, refaktoryzacja.
- **Twórcy treści** — hooki, scenariusze shortów, karuzele, serie na Instagram i TikTok.

Nie musisz znać prompt engineeringu. Wystarczy darmowa aplikacja PrompBase i ta paczka.

### Co jest w środku (250 promptów)

| Kategoria | Liczba | Przykłady |
|-----------|--------|-----------|
| Marketing i sprzedaż | 100 | Persona klienta, lead magnety, maile powitalne, landing page, cold email, SEO |
| Programowanie i DevOps | 80 | Code review, API REST, testy, Docker, CI/CD, security |
| Content i social media | 70 | Hooki Reels/TikTok, karuzele IG, scenariusze video, recykling treści |

Każdy prompt ma pola do uzupełnienia, np. `[OPIS_BIZNESU]`, `[TEMAT]`, `[JEZYK]` — w aplikacji wypełnisz je przed skopiowaniem do AI.

### Jak to działa (3 kroki)

1. Pobierz paczkę ZIP i rozpakuj.
2. Otwórz **darmowy** PrompBase: https://github.com/zetmar-collab/PrompBase-Python/releases  
3. **Plik → Szybki import JSON** → wybierz `premium-prompbase-v2.5.json` → gotowe.

Szczegóły krok po kroku są w pliku `INSTRUKCJA-IMPORT.txt` w paczce.
