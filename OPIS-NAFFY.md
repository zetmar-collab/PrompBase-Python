# PrompBase Python v2.0

## Tytuł produktu

PrompBase Python v2.0 - darmowa biblioteka promptów AI na Windows i macOS

## Krótki opis

Darmowa aplikacja desktopowa do zapisywania, porządkowania, wyszukiwania i eksportowania promptów AI. Działa lokalnie na Windows i macOS, bez konta, bez abonamentu i bez wysyłania danych na zewnętrzny serwer.

## Długi opis

PrompBase Python v2.0 to prosta, lokalna biblioteka promptów AI stworzona dla osób, które pracują z ChatGPT, Claude, MidJourney, narzędziami no-code, automatyzacjami n8n albo własnymi workflow AI.

Zamiast trzymać najlepsze prompty w notatkach, dokumentach, czatach i przypadkowych plikach, możesz zebrać je w jednej aplikacji. PrompBase pozwala dodać prompt, opisać go modelem AI, zastosowaniem, statusem i komentarzem, a potem szybko go odnaleźć, skopiować albo wyeksportować.

Aplikacja działa lokalnie na komputerze. Dane zapisują się w pliku JSON na Twoim urządzeniu. Nie potrzebujesz logowania, internetu ani serwera.

## Co dostajesz

- aplikację PrompBase Python v2.0
- wersję dla Windows z plikiem startowym `.bat`
- wersję dla macOS z plikiem `.command`
- ikony aplikacji dla Windows `.ico` i macOS `.icns`
- import i eksport CSV
- lokalny zapis danych w JSON
- integrację z webhookiem n8n
- tryb jasny i ciemny
- instrukcję uruchomienia

## Najważniejsze funkcje

- Dodawanie, edycja i usuwanie promptów
- Wyszukiwanie pełnotekstowe po nazwie, treści, komentarzu, modelu i zastosowaniu
- Filtrowanie po statusie, modelu AI i zastosowaniu
- Sortowanie: najnowsze, najstarsze, A-Z, Z-A
- Przypinanie najważniejszych promptów
- Kopiowanie treści promptu do schowka
- Import i eksport CSV
- Wysyłka pojedynczego promptu lub całej biblioteki do n8n
- Tryb jasny i ciemny
- Dane przechowywane lokalnie

## Dla kogo

Dla twórców, marketerów, freelancerów, edukatorów, automatyzatorów, programistów i wszystkich osób, które regularnie używają promptów AI i chcą mieć własną uporządkowaną bazę.

## Wymagania

- Windows albo macOS
- Python 3

Na wielu komputerach Python jest już zainstalowany. Jeśli nie, można go pobrać bezpłatnie ze strony python.org.

## Jak uruchomić

Windows:

1. Rozpakuj paczkę ZIP.
2. Otwórz folder aplikacji.
3. Kliknij `Start-PrompBase-Windows.bat`.

macOS:

1. Rozpakuj paczkę ZIP.
2. Otwórz Terminal w folderze aplikacji.
3. Wpisz:

```bash
chmod +x Start-PrompBase-macOS.command
./Start-PrompBase-macOS.command
```

## Prywatność

PrompBase działa lokalnie. Twoje prompty są zapisywane na Twoim komputerze i nie są wysyłane do żadnego zewnętrznego serwera. Wyjątkiem jest tylko ręczne użycie funkcji n8n, jeśli samodzielnie wpiszesz adres webhooka.
